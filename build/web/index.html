Sistema de Ganancias

Acerca de

GANATRANS

Sistema Informatico orientado a la web que mejora el desempeño en el rendimiento y calidad de los servicios de los conductores, ayudandolos a cumplir sus objetivos misionales permitiendoles conocer sus ganancias en tiempo real.

Ya no necesitas pensar demasiado para poder calcular tus ganancias porque GANATRANS te lo falicita todo.

Funcionalidades

Funcionalidades de GANATRANS

    El sistema puede registrar a los usuarios


    El sistema genera reportes de las ganacias en tiempo real.


    El sistema registra vehiculos y tambien se pueden actualizar.


    El sistema tambien permite registrar los gastos de cada vehiculo.


    El sistema calcula las ganacias en tiempo real de cada conductor.

Ventajas

Ventajas de GANATRANS


    Podra consultar reportes sin importar el mes ni la fecha.


    Se Puede consultar los vehiculos registrados por el número de placa.


    Podra consultar los conductores registrados por el número de documento.


    Con este sistema puede llevar un control de horario y rutas asignadas.
